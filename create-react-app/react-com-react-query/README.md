## Projeto Usando React Query (Stanstack Query)

O projeto simples com um CRUD usando react query (stanstack query).

useQuery para consultas, e useMutation para ações com alterações no banco.

O staleTime foi configurado para 30 segundos, então foi definido uma invalidação da key de lista de author na hora de mutação.

Em alguns locais utilizei hooks.

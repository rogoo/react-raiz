## React Router utilizando Declarative mode

Um projeto bem simples utilizando o **modo Declarative** do react **router**.

Use este modo para aprender ou para projetos SPA bem simples ou que serão utilizados em uma intranet - projeto de internet não é muito legal pois vai perder SEO.

**_Importante:_** Neste modo não existe typesafety ou a possibilidade de utilizar SSR.

No componente LinkUm foi utilizado parametro no link (:nome), e utilizado o hook useParams para pegar a informação.
